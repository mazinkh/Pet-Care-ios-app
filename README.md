# 🐾 Pet Care iOS App

An iOS application built in Swift that helps users manage and track their pets’ care information including appointments, profiles, and reminders.

## 📱 Features

- Create and manage pet profiles
- Track appointments and care schedules
- Organized and user-friendly interface
- Unit and UI testing included

## 🛠 Tech Stack

- Swift
- Xcode
- UIKit / SwiftUI (whichever you used)
- iOS Simulator
- XCTest for testing

## 🚀 How to Run

1. Clone the repository:
   ```bash
   git clone git@github.com:mazinkh/Pet-Care-ios-app.git
   
2.    Open Pet Care.xcodeproj in Xcode

3.    Select an iPhone simulator

4.    Press Run ▶️
